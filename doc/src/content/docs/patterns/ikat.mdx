# Ikat

Woven textile pattern with noise-displaced edges that simulate the characteristic dye bleeding of real ikat fabric, where yarns are resist-dyed before weaving.

## Algorithm

1. **Background fill** — The canvas is filled with the first palette color.

2. **Noise initialization** — A 2D simplex noise function is created from the PRNG seed, providing deterministic but organic-looking displacement values across the canvas.

3. **Vertical band drawing** — Up to 4 shuffled foreground colors are selected as band colors. Bands are drawn as vertical stripes that tile horizontally. Each band is rendered row-by-row (in 2-pixel increments) rather than as a single rectangle:
- **Horizontal displacement:** Each row segment is shifted left/right by `noise(x, y) * displacementAmount`, creating the wavy, imprecise edges characteristic of ikat.
- **Width variation:** Band width also varies per row using a second noise sample, so edges on both sides undulate independently.
- **Alpha variation:** Each segment's opacity fluctuates (0.5–0.9) based on a third noise sample at a lower frequency, simulating uneven dye saturation.

4. **Diamond/lozenge overlay** — Ikat textiles commonly feature diamond (rhombus) motifs. These are drawn as 32-segment polygons where each vertex is displaced by noise:
- The base diamond shape is traced parametrically in four quadrants (top→right→bottom→left).
- Each vertex receives independent horizontal and vertical noise displacement, producing the characteristic fuzzy-edged diamond seen in real ikat.
- Diamonds are arranged in a grid with alternating row offsets (brick-like stagger) for a balanced composition.

5. **Horizontal thread lines** — Thin semi-transparent horizontal lines are drawn every 4 pixels across the entire canvas, simulating the visible weft threads of a woven textile.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Band Width | 5–40 | 20 | Controls vertical band width as a divisor of canvas size. Higher values produce narrower bands. |
| Blur Amount | 0–1 | 0.4 | Intensity of noise-based edge displacement. Higher values produce more dye bleeding and fuzzier edges. |
| Diamond Height | 1–5 | 2.5 | Aspect ratio multiplier for diamond motifs. Higher values produce taller, more elongated diamonds. |

## Visual Characteristics

Ikat reproduces the distinctive "bleeding" look of resist-dyed woven textiles. Unlike patterns with crisp geometric edges, every boundary in this pattern is intentionally imprecise — edges wobble, colors bleed across boundaries, and shapes have an organic, hand-crafted quality. The vertical band structure references the warp-ikat technique where warp threads are dyed before weaving, producing vertical color zones with fuzzy lateral boundaries. The diamond motifs with their noisy outlines are a hallmark of Central Asian and Southeast Asian ikat traditions, where complex geometric designs emerge from the interplay of dyed and undyed thread segments. The subtle horizontal thread lines reinforce the woven-fabric reading of the pattern.
