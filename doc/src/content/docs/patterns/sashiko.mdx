# Sashiko

Japanese stitching patterns rendered as dashed lines on a fabric-textured background, supporting four traditional motifs.

## Algorithm

Sashiko (刺し子, literally "little stabs") is a form of Japanese decorative reinforcement stitching originating in the Edo period. Originally a practical technique for mending and strengthening cloth, it evolved into an art form with dozens of named geometric motifs stitched in white thread on indigo fabric.

The algorithm works as follows:

1. **Fabric background**: The canvas is filled with a slightly lightened background color. Random single-pixel dots are scattered across the surface to simulate fabric texture.

2. **Stitch rendering system**: A reusable "stitcher" function manages the dashed-line appearance. It cycles through 1–3 thread colors and optionally introduces gap irregularity — random variation in the dash gap length that mimics the imprecision of hand-stitching.

3. **Motif selection**: One of four traditional motifs is chosen, each with its own drawing function:

   - **Asa-no-ha (hemp leaf)**: For each grid cell, 6 lines radiate from the center at 60° intervals, forming a six-pointed star. A diamond outline (rotated square) is drawn around each cell. The combined effect creates the classic hemp-leaf tessellation.

   - **Nowaki (waves)**: Rows of concentric semicircular arcs, offset alternately by half a cell width. Each position draws 3 nested arcs of increasing radius, creating the layered wave pattern also known as seigaiha.

   - **Uroko (scales)**: An equilateral triangle grid where each position produces one upward and one downward triangle. The triangular tessellation resembles fish scales or dragon scales.

   - **Yabane (arrow feathers)**: Chevron/arrow shapes arranged in offset rows. Each arrow consists of left and right diagonal halves meeting at a center point, plus horizontal lines subdividing the interior.

4. **Motif rotation**: The entire motif layer can be rotated by 0°, 90°, 180°, or 270°. For 90°/270° rotations, extra cells are generated to ensure full canvas coverage after the transform.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Motif | Asa-no-ha, Nowaki, Uroko, Yabane | Asa-no-ha | Which traditional sashiko motif to render. |
| Cell Size | 4–30 | 12 | Grid division count along the longest axis. Lower values produce larger motifs. |
| Stitch Dash | 0.05–0.3 | 0.15 | Dash length as a fraction of cell size. Controls the visible stitch length. |
| Thread Colors | 1–3 | 1 | Number of distinct thread colors drawn from the palette. Multiple colors cycle across stitches. |
| Gap Irregularity | 0–1 | 0 | Randomness in the gap between dashes. At 0, gaps are uniform; at 1, they vary significantly, simulating hand-stitching. |
| Thread Thickness | 0.5–3.0 | 1.0 | Multiplier for the line width of each stitch. |
| Motif Rotation | 0°, 90°, 180°, 270° | 0° | Rotation applied to the entire motif layer. |

## Visual Characteristics

The dashed-line rendering is the defining visual feature — it distinguishes sashiko from solid-line geometric patterns and immediately evokes the feel of needle and thread on cloth. The fabric texture background reinforces this textile quality.

Asa-no-ha is the most iconic and dense motif, filling the surface evenly. Nowaki produces flowing, organic curves that contrast with the angular geometry of the other three. Uroko has a distinctive triangular rhythm. Yabane creates strong directional movement with its arrow shapes.

Gap irregularity is a key expressive parameter: at 0 the stitches look machine-precise, while higher values add a hand-crafted, wabi-sabi quality. Multiple thread colors create rich visual rhythm as different hues alternate across the stitch lines.
