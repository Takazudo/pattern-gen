# Guilloche

Spirograph-style engraving patterns drawn from parametric hypotrochoid curves — the intricate, interlocking line work seen on banknotes and security documents.

## Algorithm

Guilloche patterns are produced by mechanical lathes (or their mathematical equivalents) that trace complex periodic curves. This implementation draws **hypotrochoid** curves — the path traced by a point attached to a smaller circle rolling inside a larger one.

### Hypotrochoid Equations

For a fixed outer circle of radius **R**, an inner circle of radius **r** rolling inside it, and a pen point at distance **d** from the inner circle's center:

- `x(t) = (R − r)·cos(t) + d·cos(t·(R − r)/r)`
- `y(t) = (R − r)·sin(t) − d·sin(t·(R − r)/r)`

where `t` is the angle parameter.

### Parameter Selection

For each curve (2–4 overlaid curves):

- **R** (outer radius): Randomly scaled between 60%–100% of the maximum drawing radius.
- **r** (inner radius): `R / rDivisor`, where `rDivisor` is a random integer from 2 to 9. Using integer divisors ensures the curve closes cleanly.
- **d** (pen distance): `r · penDistanceFactor`. When `d < r`, the curve stays inside the outer circle; when `d = r`, it touches the outer circle (producing cusps); when `d > r`, it extends beyond.

### Curve Completion

A hypotrochoid completes (returns to its starting point) after `LCM(R, r) / r` revolutions of the inner circle. The implementation computes this using the GCD (Euclidean algorithm) on scaled integer representations of R and r, then derives the total angle range: `totalAngle = revolutions · 2π`.

### Drawing

The curve is sampled at up to ~8000 points (scaled by canvas size) and rendered as a single continuous `Path2D` stroke. Each overlaid curve uses a different palette color with 50%–80% opacity, so intersections create rich color blending.

## Parameters

| Parameter    | Range   | Default  | Description                                        |
| ------------ | ------- | -------- | -------------------------------------------------- |
| Curve Count  | 2 – 4   | 2        | Number of overlaid hypotrochoid curves             |
| Line Width   | 0.3 – 2 | 0.8      | Stroke width of each curve                         |
| Pen Distance | 0.2 – 1.2 | 0.7    | Ratio of pen offset to inner circle radius         |

## Visual Characteristics

Produces the delicate, interlocking line work characteristic of banknote engravings, stock certificates, and Spirograph toys. The curves exhibit perfect rotational symmetry determined by the ratio R/r — an integer divisor of 3 produces 3-fold symmetry, 7 produces 7-fold, and so on. Overlaying multiple curves with different parameters creates dense, layered filigree patterns. The mathematical precision of these curves historically made them difficult to forge, which is why they became a hallmark of security printing.
