# Flow Field

Thousands of small line segments follow noise-derived angles across the canvas, creating organic visuals resembling wind currents, hair strands, or magnetic field lines.

## Algorithm

The algorithm places particles at random positions and traces their paths through a 2D simplex noise field:

1. A 2D simplex noise function is initialized from the deterministic PRNG seed.
2. For each of the `lineCount` particles, a random start position `(x, y)` is chosen on the canvas.
3. Each particle is assigned a random foreground color, alpha (0.3–0.8), and line width.
4. The particle steps forward `stepsPerLine` times. At each step:
   - The noise value at the current position is sampled: `noise(x * noiseScale, y * noiseScale)`.
   - The value is multiplied by `2π` to produce an angle in radians.
   - The particle moves 2 pixels in the direction of that angle: `x += cos(angle) * 2`, `y += sin(angle) * 2`.
   - If the particle exits the canvas bounds, the line terminates early.
5. The accumulated path is stroked as a single connected line with round caps.

The `noiseScale` parameter controls the spatial frequency of the noise field — lower values produce smoother, more sweeping curves while higher values create tighter, more chaotic flow.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Noise Scale | 0.001 – 0.01 | 0.003 | Spatial frequency of the noise field. Lower = smoother flow, higher = more turbulent |
| Line Count | 500 – 5000 | 2000 | Number of particle lines drawn |
| Steps per Line | 5 – 80 | 35 | Maximum length of each particle trace (in 2px steps) |
| Line Width | 0.3 – 3.0 | 1.0 | Maximum stroke width of each line |

## Visual Characteristics

Flow Field produces a distinctly organic, directional texture. Dense clusters of parallel curves emerge in regions where the noise field is smooth, while areas of rapid noise change create swirling eddies. The semi-transparent overlapping strokes build up rich tonal depth, with foreground colors blending where paths converge. The overall effect is reminiscent of wind maps, fingerprint whorls, or Van Gogh's Starry Night brushwork.
