# Stipple

Pointillist dot-matrix pattern that uses noise-driven density fields to place dots of varying size, creating an effect reminiscent of stipple engraving or newspaper halftone printing.

## Algorithm

1. **Noise field setup**: Create a 2D simplex noise function seeded by the deterministic PRNG.
2. **Grid construction**: Lay out a regular grid with spacing `dotSpacing / zoom`. Each grid intersection becomes a candidate dot position.
3. **Position jittering**: Offset each grid point by a random displacement scaled by the `jitter` parameter (0 = perfect grid, 1 = maximum randomness within one cell width).
4. **Density sampling**: For each jittered position, evaluate 4-octave fractal Brownian motion (FBM) at the noise-scaled coordinates. Normalize the result from [-1, 1] to [0, 1] to get a local density value.
5. **Radius mapping**: Map the density value linearly to a dot radius between `minRadius` (8% of spacing) and `maxRadius` (45% of spacing). Skip dots smaller than 0.5px.
6. **Color selection**: Sample a second, offset noise field (2× frequency, shifted by +100 in both axes) to determine dot color. The noise value maps to an index into the shuffled foreground palette.
7. **Dot rendering**: Draw each dot as a filled circle (`arc` + `fill`) at the jittered position.

The dual-noise approach — one field for size, another for color — ensures that density and hue vary independently across the canvas, producing rich visual texture.

## Parameters

| Parameter | Range | Default | Description |
| --- | --- | --- | --- |
| Dot Spacing | 3–20 | 8 | Distance between grid points in pixels (controls dot density) |
| Noise Scale | 0.001–0.02 | 0.005 | Frequency of the density noise field (higher = more variation) |
| Jitter | 0–1 | 0.6 | Random offset applied to grid positions (0 = regular, 1 = scattered) |

## Visual Characteristics

- Dense, organic dot patterns that evoke pointillist art or stipple illustration techniques
- Dot sizes vary smoothly across the canvas following the noise field — areas of high density have large dots, low density areas have small or absent dots
- The jitter parameter controls regularity: low values produce a mechanical dot-matrix look, high values create a hand-placed, organic feel
- Colors shift gradually across regions due to the independent color noise field
- Smaller dot spacing creates finer, more detailed textures; larger spacing produces bolder, more graphic compositions
