# Marble

Marble veining using the classic turbulent noise technique, producing realistic stone-like patterns with flowing colored veins.

## Algorithm

The pattern uses the well-known marble formula: `sin(position * frequency + turbulence * amplitude)`, where turbulence is provided by fBm noise:

1. A 2D simplex noise function is initialized from the deterministic seed.
2. A random vein direction angle (0–π) is chosen per seed. This determines the primary axis along which veins flow, via direction vector `(cos(angle), sin(angle))`.
3. For each pixel `(x, y)`:
   - **Turbulence layers**: Two independent fBm evaluations produce turbulence values. The first uses 6 octaves at the base noise position. The second uses 4 octaves at an offset position `(+3.7, +7.1)` for decorrelation.
   - **Projection**: The pixel position is projected along the vein direction: `projected = (x * dirX + y * dirY) * veinFrequency`.
   - **Marble formula**: `sin(projected + turb1 * amplitude + turb2 * amplitude * 0.5)` combines the directional base frequency with turbulent displacement.
4. The sine output `[-1, 1]` is normalized to `[0, 1]`, then passed through a power curve (`pow(n, 0.6)`) to sharpen the vein structure.
5. The vein intensity blends between the background color and a foreground vein color (selected per-pixel based on the second turbulence value).
6. The result is a smooth interpolation between background and vein colors, rendered via `ImageData`.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Noise Scale | 0.002 – 0.015 | 0.005 | Spatial frequency of the turbulence noise |
| Vein Frequency | 0.005 – 0.1 | 0.02 | Frequency of the base sine wave that forms the veins |
| Turbulence Amplitude | 1 – 20 | 5 | How much the turbulence displaces the veins. Higher = more chaotic |

## Visual Characteristics

Marble produces flowing, organic veins on a stone-like ground — closely mimicking real polished marble. The veins follow a primary direction (randomized per seed) but are displaced by turbulence into natural-looking curves and forks. The dual turbulence layers create both large-scale vein wandering and fine-grained texture. The power curve sharpening produces distinct vein lines rather than blurry gradients. Multiple foreground colors create the characteristic multi-toned veining seen in natural stone.
