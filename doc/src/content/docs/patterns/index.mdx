# Pattern Catalog

pattern-gen includes 30 distinct pattern generation algorithms. Each produces deterministic output from a text seed and supports all 36 color schemes.

## Grid & Tile Patterns

| Pattern | CLI Name | Description |
|---------|----------|-------------|
| [Wood Block](/docs/patterns/wood-block) | `wood-block` | Diamond grid with noise-based color gradients forming mountain shapes |
| [Truchet](/docs/patterns/truchet) | `truchet` | Quarter-circle arc tiles creating flowing organic curves |
| [Herringbone](/docs/patterns/herringbone) | `herringbone` | V-shaped zigzag brick pattern (parquet flooring) |
| [Isometric](/docs/patterns/isometric) | `isometric` | 3D cube grid with face shading (Escher-like illusion) |
| [Penrose](/docs/patterns/penrose) | `penrose` | Aperiodic P3 kite-and-dart tiling |
| [Patchwork](/docs/patterns/patchwork) | `patchwork` | Quilt blocks (log cabin, nine-patch, pinwheel) |
