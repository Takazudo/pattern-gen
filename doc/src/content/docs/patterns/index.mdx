# Pattern Catalog

pattern-gen includes 30 distinct pattern generation algorithms. Each produces deterministic output from a text seed and supports all 36 color schemes.

## Grid & Tile Patterns

| Pattern | CLI Name | Description |
|---------|----------|-------------|
| Wood Block | `wood-block` | Diamond grid with noise-based color gradients forming mountain shapes |
| Truchet | `truchet` | Quarter-circle arc tiles creating flowing organic curves |
| Herringbone | `herringbone` | V-shaped zigzag brick pattern (parquet flooring) |
| Isometric | `isometric` | 3D cube grid with face shading (Escher-like illusion) |
| Penrose | `penrose` | Aperiodic P3 kite-and-dart tiling |
| Patchwork | `patchwork` | Quilt blocks (log cabin, nine-patch, pinwheel) |

## Noise & Flow Patterns

| Pattern | CLI Name | Description |
|---------|----------|-------------|
| Flow Field | `flow-field` | Simplex noise-driven particle flow lines |
| Domain Warp | `domain-warp` | Layered noise with warping for organic shapes |
| Topographic | `topographic` | Contour map lines from noise height field |
| Marble | `marble` | Turbulent sine-warped noise for marble veining |
| Wave Interference | `wave-interference` | Overlapping sine wave sources creating interference rings |
