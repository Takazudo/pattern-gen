# Chevron

Zigzag V-shaped stripes with smooth color gradients between adjacent rows, producing a continuous chevron weave effect.

## Algorithm

1. **Background fill** — The canvas is filled with the first palette color.

2. **Color sequence** — Foreground palette colors are shuffled and cycled to create enough entries to cover all rows. Each row is assigned a color from this repeating sequence.

3. **Row wobble** — Each row gets a random wobble factor (ranging from approximately 0.9x to 1.1x of base row height, controlled by a seed-based wobble strength of 0.05–0.15). This introduces slight height variation between rows, preventing a mechanical, uniform appearance.

4. **Gradient subdivision** — Each chevron row is subdivided into multiple gradient steps. For each step, the color is linearly interpolated (`lerpColor`) between the current row's color and the next row's color. When `gradientSteps` is set to 0 (default), the step count is computed automatically as `max(4, rowHeight / 3)`.

5. **V-shape rendering** — Each gradient sub-row is drawn as a filled polygon forming a V-shape (chevron). The polygon has six vertices: three along the top edge of the sub-row and three along the bottom edge. Each edge traces the V — starting at the left edge, peaking at the canvas center (x = width/2), and descending to the right edge. The vertical offset at the left and right edges equals one full wobbled row height below the center peak.

6. **Cumulative positioning** — Rows are positioned using cumulative Y tracking (rather than fixed offsets) to correctly account for wobbled heights, ensuring rows seamlessly stack without gaps.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Row Height | 4–30 | 16 | Controls chevron row height as a divisor of canvas size. Higher values produce thinner rows. |
| Gradient Steps | 0–20 | 0 | Number of color interpolation steps between adjacent rows. 0 = automatic (based on row height). Higher values produce smoother gradients. |

## Visual Characteristics

Chevron produces continuous zigzag stripes that resemble herringbone or chevron-woven fabric. The gradient interpolation between adjacent rows creates smooth color transitions rather than hard edges, giving the pattern a rich, textile-like depth. The per-row wobble factor adds organic irregularity — rows are not perfectly uniform, mimicking the slight tension variations found in hand-woven chevron cloth. The V-shapes always peak at the horizontal center of the canvas, creating a strong vertical axis of symmetry.
