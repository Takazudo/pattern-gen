# Zellige

Moroccan mosaic tilework with hand-cut irregular shapes, visible grout lines, and a glazed ceramic finish.

## Algorithm

Zellige (also spelled zellij) is a form of Islamic mosaic tilework made from individually hand-chiseled geometric tile pieces set in plaster. Originating in 10th-century Morocco, the craft involves cutting small glazed terracotta pieces into precise shapes and assembling them into intricate patterns. Each tile carries subtle color variation from the glazing process.

The algorithm works as follows:

1. **Grout base layer**: The entire canvas is filled with a darkened version of the background color, representing the mortar or plaster visible between tiles.

2. **Jittered grid generation**: A regular grid of points is created, then each point is displaced randomly by a jitter amount (up to `cellSize * jitterFactor` in each direction). This transforms the rigid grid into an irregular lattice that mimics the imprecision of hand-cut tiles. An extra ring of points beyond the canvas edge ensures full coverage.

3. **Tile construction**: Adjacent grid points form irregular quadrilaterals — each tile uses four neighboring jittered points (top-left, top-right, bottom-right, bottom-left). Each tile is then inset toward its centroid by the grout width factor, creating visible gaps that represent the mortar lines.

4. **Glazed ceramic coloring**: Each tile receives a base color randomly selected from the foreground palette, then modified by two factors:
   - A per-tile random variation (0.9–1.1 multiplier) simulating kiln temperature differences
   - A Simplex noise-based modulation (±8%) creating smooth, regional color drift across the surface

5. **Glaze highlights**: A linear gradient from top to bottom of each tile adds a glaze sheen — a lighter highlight at the top fading to transparent in the middle, then a subtle darkening at the bottom edge. Top edges receive a light stroke and bottom edges a dark stroke for a 3D beveled appearance.

6. **Surface imperfections**: A sparse scattering of small circles in the grout color simulates chips and wear in the glaze, adding to the handmade character.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Cell Size | 5–40 | 16 | Grid division count along the longest axis. Lower values produce larger, fewer tiles. |
| Jitter | 0–0.6 | 0.2 | How much grid points are randomly displaced, as a fraction of cell size. 0 produces perfectly regular tiles; 0.6 creates highly irregular shapes. |
| Grout Width | 0.02–0.2 | 0.06 | Width of the visible mortar gap between tiles, as a fraction of cell size. |

## Visual Characteristics

The pattern produces a surface that closely resembles real zellige tilework, with its characteristic irregularity and warm, glazed color variation. The jitter parameter is the key differentiator from other tile patterns — even small values create the subtle imprecision that makes zellige instantly recognizable as handmade rather than machine-cut.

At low jitter (0–0.1), the tiles are nearly rectangular with minor wobble. At moderate jitter (0.2–0.3), tiles become clearly irregular quadrilaterals with a natural, organic feel. At high jitter (0.4–0.6), tiles become dramatically distorted, creating an almost abstract mosaic.

The grout width controls visual density — narrow grout (0.02–0.04) creates a tightly packed surface where color dominates, while wide grout (0.1–0.2) emphasizes the grid structure and reveals more of the dark mortar background. The glazed ceramic finish — with per-tile color shifts, top-light/bottom-dark gradients, and occasional chips — adds material realism that distinguishes this from flat color fills.
