# Op Art

Riley-style optical art with concentric shapes whose spacing is sinusoidally modulated to create the illusion of a 3D surface with convex bumps or concave dips.

## Algorithm

Inspired by Bridget Riley's optical paintings, this pattern draws concentric circles or squares from a central point, varying the spacing between rings to simulate perspective distortion on a curved surface.

### Bump Points

1–2 "bump" focal points are placed randomly within the central 60% of the canvas. Each bump has a position `(x, y)` and a strength parameter that controls the intensity of the spatial distortion.

### Ring Generation

Starting from the center, concentric shapes are drawn outward with dynamically modulated spacing:

1. **Spacing modulation**: For each ring at radius `currentRadius`, the spacing is adjusted based on proximity to each bump point:
   - Distance from ring to bump: `dist = |currentRadius − √((cx − bump.x)² + (cy − bump.y)²)|`
   - Gaussian influence: `influence = e^(−dist² / (maxDim² · 0.02))`
   - Modulation: `spacingMod += strength · sin(dist · 0.05) · influence`
   - The sinusoidal term creates alternating compression and expansion of rings near the bump, while the Gaussian envelope limits the effect to the bump's vicinity.
2. **Final spacing**: `spacing = baseSpacing · clamp(spacingMod, 0.3, 2.5)` — clamped to prevent degenerate cases.
3. **Radius advancement**: `currentRadius += spacing`. This accumulation means spacing variations compound, creating visible distortion zones.

### Shape Drawing

- **Circles**: Drawn as arcs with `ctx.arc()`.
- **Squares**: Drawn as concentric `strokeRect()` calls centered on the canvas.
- Colors alternate between two randomly selected palette colors. Line width is set to `spacing · 0.45` so rings nearly touch, maximizing the optical density.

## Parameters

| Parameter     | Range     | Default | Description                                      |
| ------------- | --------- | ------- | ------------------------------------------------ |
| Shape Type    | —         | Circles | Concentric circles or squares                    |
| Base Spacing  | 1 – 15    | 6       | Base distance between concentric shapes (px)     |
| Bump Strength | 0.1 – 1.5 | 0.55   | Intensity of the 3D surface distortion illusion   |

## Visual Characteristics

Creates striking optical illusions where flat concentric shapes appear to bulge outward or recede inward, as if painted on a warped surface. The effect exploits the visual system's tendency to interpret spacing gradients as depth cues. Tighter spacing reads as a surface curving away; wider spacing reads as curving toward the viewer. With two bump points, the pattern produces a more complex, undulating surface. The two-color alternation maximizes contrast, enhancing the perceptual distortion.
