# Penrose

Aperiodic Penrose P3 tiling generated through recursive subdivision, producing a non-repeating pattern with five-fold rotational symmetry using kite and dart shapes.

## Algorithm

The pattern implements the Robinson triangle decomposition method for generating Penrose P3 tilings:

1. **Initial decagon**: The algorithm starts by constructing a decagon of 10 isosceles triangles radiating from the canvas center, alternating orientation. These are all "thick" triangles (type 1), which will subdivide into the kite halves of the final tiling. The decagon radius is scaled to cover the canvas.

2. **Recursive subdivision**: Each triangle is recursively subdivided using the golden ratio φ = (1 + √5) / 2:
   - **Thick triangles** (kite halves) split into one thick and one thin triangle. A new vertex is placed along one edge at a distance of 1/φ from the apex.
   - **Thin triangles** (dart halves) split into one thick and two thin triangles. Two new vertices are placed using the golden ratio along different edges.
   This subdivision preserves the aperiodic property — the pattern never repeats, no matter how many times it is subdivided. The number of subdivisions is controlled by the Subdivisions parameter.

3. **Color mapping**: Kite halves and dart halves receive different color pairs from the palette. Within each pair, the exact color is interpolated based on the triangle's position on the canvas (using the centroid's normalized x+y coordinate), creating a smooth gradient across the tiling. A small random brightness variation is applied per triangle for visual texture.

4. **Edge rendering**: Each triangle is outlined with a darkened stroke at configurable width, defining the tile boundaries and enhancing the geometric structure.

5. **Deterministic randomness**: Random values for brightness variation are consumed for every triangle (including off-screen ones) before the visibility cull, ensuring the pattern remains deterministic across different canvas sizes and zoom levels.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Subdivisions | 2–8 | 5 | Number of recursive subdivision passes. Each level roughly triples the triangle count, producing finer detail. |
| Line Width | 0.3–3 | 1 | Stroke width of triangle edges. Thicker lines emphasize the tiling structure. |
| Edge Darkness | 0.3–1.0 | 0.7 | How dark the edge lines are relative to the fill color. |

## Visual Characteristics

The Penrose tiling is mathematically fascinating — it fills the plane completely without ever repeating. The pattern exhibits five-fold rotational symmetry with local star and sun motifs emerging naturally from the subdivision. At low Subdivisions (2–3), the triangles are large and the pentagonal symmetry is clearly visible as a bold geometric design. At high subdivisions (6–8), the tiles become very fine and the pattern takes on an intricate, crystalline quality where the aperiodic nature is most apparent. The position-based color gradient creates a subtle directional flow across the tiling, while the per-triangle brightness variation prevents the surface from looking flat. Thick edge lines give the pattern a stained-glass quality, while thin lines let the color fields dominate. The golden ratio geometry means that the same structural motifs appear at every scale but never in exactly the same arrangement.
