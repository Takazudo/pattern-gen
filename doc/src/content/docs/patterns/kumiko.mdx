# Kumiko

Japanese kumiko geometric woodwork — layered triangle grid patterns with multiple traditional sub-patterns overlaid with rotation and translation.

## Algorithm

Kumiko (組子) is a Japanese woodworking technique where thin strips of wood are precisely cut and assembled without nails or glue into intricate geometric lattice patterns. Traditionally used in shoji screens, transoms (ranma), and furniture, kumiko patterns are based on an equilateral triangle grid that serves as the foundation for dozens of named sub-patterns.

The algorithm works as follows:

1. **Equilateral triangle grid**: A grid of equilateral triangles is generated across an oversized canvas (1.5× the actual dimensions to allow for rotation without gaps). Each column produces one upward-pointing triangle and one downward-pointing triangle. The grid records each triangle's three vertices, centroid, and orientation.

2. **Layer selection**: Up to 4 layers are drawn, each assigned a randomly selected sub-pattern from the available set. The sub-patterns are shuffled and the first `layerCount` are chosen, ensuring variety across layers.

3. **Six sub-patterns** are available, each defining a different way to subdivide the base triangles:

   - **Asanoha**: Lines from each vertex to the triangle's centroid, creating the classic hemp-leaf star radiating pattern.
   - **Mitsukude**: Lines from each edge midpoint to the centroid, forming a three-pronged inner star.
   - **Goma**: Lines from each vertex to the midpoint of the opposite edge (medians), meeting at the centroid.
   - **Shippo**: Circular arcs connecting adjacent vertices with a radius of half the edge length, creating interlocking circles (seven treasures pattern).
   - **Kikko**: An inner triangle formed at 40% lerp from centroid toward each vertex, with lines connecting vertices to their corresponding inner points, creating a tortoiseshell effect.
   - **Yae-asanoha**: A 6-spoke star combining both vertex-to-centroid and midpoint-to-centroid lines, creating a denser version of asanoha.

4. **Layer compositing**: Each layer is drawn with a random rotation angle (0–360°), random translation offset (±20% of canvas size), and its own color from the palette. This creates visual depth as multiple geometric networks intersect at different angles.

5. **Overlap copies**: Each layer can optionally render 1–3 slightly offset copies with small random rotation and translation deltas, producing a moiré-like density effect.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Grid Divisions | 4–16 | 8 | Number of triangle columns across the canvas. Adjusted by zoom. Lower values produce larger triangles. |
| Layer Count | 1–4 | 3 | Number of distinct sub-pattern layers overlaid on each other. |
| Stroke Width | 0.5–5 | 2 | Line width for all drawn strokes, adjusted by zoom. |
| Overlap Copies | 1–3 | 1 | Number of slightly offset copies per layer. Values above 1 create denser, moiré-like overlaps. |

## Visual Characteristics

The pattern produces intricate, lattice-like geometric fields that evoke the precision of traditional Japanese woodwork. The equilateral triangle grid provides a strong underlying order, while the random rotation and offset of each layer creates rich visual complexity from relatively simple component patterns.

With a single layer, the pattern is clean and readable — the chosen sub-pattern is clearly visible in its triangular grid. At 2–3 layers, the overlapping rotated grids create beautiful interference patterns where different geometries intersect. At 4 layers, the result becomes dense and textile-like.

The sub-patterns offer distinct visual characters: asanoha and yae-asanoha are star-like and radial, mitsukude and goma produce angular subdivisions, shippo introduces curves with its circular arcs, and kikko creates a nested, hexagonal feel. The overlap copies parameter adds subtle density without the complexity of another distinct pattern — useful for creating a slightly textured look at overlap count 2 or a rich layered effect at 3.
