# Moiré

Interference patterns generated by overlaying parallel line grids at slight angular offsets, producing sweeping visual bands where lines constructively and destructively overlap.

## Algorithm

Moiré patterns emerge when two or more periodic structures (gratings, grids, meshes) are superimposed with a small difference in angle or spacing. This implementation draws semi-transparent line layers that optically interfere.

### Layer Construction

For each layer (2 or 3 total):

1. **Angle**: The first layer uses a random base angle. Subsequent layers are offset by a small delta: `angleDelta = ±(0.02 + rand · 0.06)`, multiplied by the layer index. This slight angular difference is what produces the interference.
2. **Spacing**: Each layer's line spacing is slightly varied: `spacing = baseSpacing · (1 + layer · 0.05 · ±1)`, introducing minor spatial frequency differences that amplify the moiré effect.
3. **Color and opacity**: Each layer uses a different palette color with alpha between 0.3 and 0.5, so overlapping regions blend visually.

### Line Drawing

For each layer, parallel lines are drawn to cover the entire canvas:

- The number of lines is computed from the canvas diagonal: `numLines = ⌈diagonal / spacing⌉ + 2`.
- Each line is offset perpendicular to the layer's angle direction, then extended across the full diagonal length to ensure complete coverage regardless of rotation.
- Line endpoints are calculated using trigonometric rotation: offset by `(i · spacing · cos(θ), i · spacing · sin(θ))` from canvas center, with line direction perpendicular to the offset.

### Interference

No explicit interference calculation is performed. The moiré effect arises naturally from the alpha-blended overlap of the line layers on the canvas — where lines from different layers coincide, the color intensifies; where they don't, the background shows through. The brain perceives these overlapping regions as large-scale wave-like bands.

## Parameters

| Parameter    | Range  | Default  | Description                          |
| ------------ | ------ | -------- | ------------------------------------ |
| Layer Count  | 2 or 3 | 2        | Number of overlaid line grids        |
| Line Spacing | 2 – 30 | 9        | Distance between parallel lines (px) |
| Line Width   | 0.5 – 5 | 1.75    | Stroke width of each line            |

## Visual Characteristics

Produces large-scale, slowly undulating bands or wave-like forms from fine-grained line grids — a pattern familiar from overlapping window screens or fabric layers. The visual effect is highly sensitive to angle and spacing differences: tiny parameter changes create dramatically different interference patterns. With 3 layers, hexagonal or more complex interference structures can emerge. The patterns have a hypnotic, optical quality related to op art.
