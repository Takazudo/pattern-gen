# Patchwork

A quilt-inspired pattern where the canvas is divided into square blocks, each filled with a randomly chosen traditional quilting sub-pattern: log cabin, nine-patch, pinwheel, or half-square triangle.

## Algorithm

The pattern generates a patchwork quilt by composing four distinct sub-patterns within a grid:

1. **Block grid**: The canvas is divided into a regular grid of square blocks. Each block is independently assigned one of four sub-patterns and its own set of random colors from the palette. Blocks are clipped to prevent sub-patterns from bleeding into neighbors.

2. **Log Cabin**: Concentric rectangular strips spiral outward from a small center square. The center is filled with one color, then alternating strips are drawn in two other colors (even strips in one, odd strips in another). Each strip has a random brightness variation applied. The resulting block has a characteristic spiral of nested rectangles.

3. **Nine-Patch**: The block is divided into a 3×3 grid. Cells at positions where (row + column) is even receive one accent color, and odd positions receive another. Each cell gets a random brightness variation and a thin inner border stroke for definition. This creates a classic checkerboard-like quilt square.

4. **Pinwheel**: The block's background is filled first, then four triangles are drawn from each corner to the center, each in a different color. The triangles are arranged so they appear to rotate around the center point, creating a windmill or pinwheel effect. Each triangle has a darkened edge stroke.

5. **Half-Square Triangle**: The block is subdivided into a 2×2 grid. Each sub-cell is split diagonally into two triangles of different colors. The diagonal direction alternates based on position `(row + col) % 2`, and colors cycle through the palette. This creates a dynamic, tessellated effect within each block.

6. **Quilt stitching**: After all sub-patterns are drawn, each block receives a border stroke in a darkened background color, simulating the seams or stitching lines of a real quilt.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Grid Divisions | 3–15 | 6 | Number of quilt blocks across the longest dimension. Fewer divisions create larger, more detailed blocks. |
| Border Width | 0.5–3 | 1.5 | Width of the "stitching" lines between blocks. |
| Color Variation | 0–0.3 | 0.1 | Maximum brightness variation within sub-patterns. Adds the subtle imperfections of hand-dyed fabric. |

## Visual Characteristics

The patchwork pattern captures the handcrafted charm of a traditional quilt. Because each block independently selects from four sub-patterns, the result is a diverse mosaic where no two blocks are typically the same. At low Grid Divisions (3–5), individual blocks are large enough to clearly see the internal structure of each sub-pattern — the spiraling strips of log cabin, the grid of nine-patch, the spinning triangles of pinwheel. At high divisions (10+), the blocks become small and the overall effect is a richly textured, colorful patchwork surface. The Border Width parameter controls the visual weight of the seams — thick borders create a strong grid structure that unifies the diverse blocks, while thin borders let the sub-patterns flow more freely. Color Variation adds warmth and character, as real quilts are rarely made from perfectly uniform fabric.
