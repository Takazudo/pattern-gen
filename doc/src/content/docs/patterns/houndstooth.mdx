# Houndstooth

Classic jagged check pattern used in woven textiles, rendered from a 4x4 matrix of filled squares and corner triangles with support for multiple variants, rotation, and multi-color modes.

## Algorithm

1. **Pattern matrix** — The core pattern is defined by a 4x4 matrix where each cell is one of: `0` (empty/background), `1` (filled square), `2` (upper-left triangle), or `3` (lower-right triangle). This matrix encodes the characteristic houndstooth "tooth" shape:

   ```
   [1, 1, 2, 0]
   [1, 1, 1, 2]
   [3, 0, 1, 1]
   [3, 1, 1, 1]
   ```

2. **Grid tiling** — The canvas is divided into a grid of cells, each sized by the `cellSize` parameter. The 4x4 pattern matrix tiles across the grid using modular indexing (`row % 4`, `col % 4`).

3. **Cell rendering** — Each cell is drawn based on its matrix value:
- **Type 1 (full):** A filled rectangle covering the entire cell.
- **Type 2 (upper-left triangle):** A triangle from the top-left corner, sized by the `toothDepth` parameter.
- **Type 3 (lower-right triangle):** A triangle from the bottom-right corner, also sized by `toothDepth`.

4. **Variant modes** — Four rendering variants modify the base pattern:
- **Classic (0):** Standard houndstooth as described above.
- **Broken (1):** Randomly skips 15% of cells and applies slight size variation (shrink up to 10%) to simulate worn or irregular fabric.
- **Stepped (2):** Replaces diagonal triangles with stair-step rectangles, approximating triangles as descending/ascending rectangular slices.
- **Diagonal (3):** Rotates the entire classic grid by 45 degrees (with 1.42x scale to fill corners), creating a diamond-oriented houndstooth.

5. **Rotation and color** — An optional 90°/180°/270° rotation is applied via canvas transform. Multi-color mode assigns different foreground colors to cell groups: top block (rows 0–1 filled squares), bottom block (rows 2–3 filled squares), and triangle cells.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Cell Size | 10–80 | 40 | Controls cell size as a divisor of canvas size. Higher values produce smaller cells. |
| Contrast | 0.3–1.0 | 0.8 | Alpha transparency of foreground cells. Lower values produce a subtler, faded pattern. |
| Tooth Depth | 0.3–1.0 | 0.7 | How far the triangle "teeth" extend into their cells. 1.0 fills the full diagonal; lower values create smaller teeth. |
| Variant | Classic / Broken / Stepped / Diagonal | Classic | Rendering style for the houndstooth grid. |
| Rotation | 0° / 90° / 180° / 270° | 0° | Rotates the entire pattern around the canvas center. |
| Color Count | 1 / 2 / 3 | 1 | Number of foreground colors. Multiple colors are assigned to different cell groups within the 4x4 tile. |

## Visual Characteristics

Houndstooth produces the distinctive "broken check" or "dogstooth" pattern — abstract four-pointed shapes that tessellate in a jagged grid. The pattern's visual identity comes from the interplay between the filled square blocks and the triangular teeth that extend from their corners, creating the illusion of interlocking shapes. The stepped variant reveals how the pattern relates to twill weave structure, where the diagonal is built from staggered horizontal picks. The broken variant adds an organic, hand-woven quality, while the diagonal variant emphasizes the underlying diamond geometry.
