# Crystal

Gem-like faceted Voronoi cells with directional light shading and inner facet lines radiating from each cell's centroid to its edges.

## Algorithm

1. **Seed point generation**: Place `seedCount × zoom²` random points across the canvas, each assigned a foreground palette color with pre-parsed RGB values.
2. **Cell map construction**: For every pixel, compute the squared Euclidean distance to all seed points and record the index of the nearest one in an `Int32Array` cell map.
3. **Cell bounds and centroid calculation**: Iterate over all pixels to accumulate per-cell bounding boxes and running sums of x/y coordinates. Compute each cell's centroid (average position of all its pixels) and maximum half-extent for distance normalization.
4. **Faceted shading (per-pixel)**: For each pixel:
   - Compute the normalized distance `t` from the cell centroid (0 at center, 1 at edge).
   - Approximate a facet normal as the unit vector from centroid to pixel.
   - Calculate light intensity via dot product of the facet normal with the light direction vector (derived from `lightAngle` in degrees).
   - Combine edge darkening (`edgeDarkening × t`) with a light boost (`0.25 × lightIntensity × (1 − 0.5t)`) to produce a net brightness adjustment.
   - Apply the adjustment to the seed's base RGB color: positive values lighten toward white, negative values darken toward black.
5. **Facet line rendering**: For each cell, scan boundary pixels at sampled intervals. Select approximately 8 evenly-spaced edge points and draw semi-transparent lines from each to the cell centroid using the darkened seed color.
6. **Border rendering**: Fill sampled boundary pixels with the darkened seed color at 60% opacity to mark cell edges.

## Parameters

| Parameter | Range | Default | Description |
| --- | --- | --- | --- |
| Seed Count | 5–100 | 30 | Base number of crystal facets (scales with zoom²) |
| Light Angle | 0–360° | 233 | Direction of the light source in degrees |
| Edge Darkening | 0.05–0.5 | 0.15 | How much cell edges darken relative to centers |

## Visual Characteristics

- Cells appear as cut gemstone facets with smooth light-to-dark gradients across each face
- Directional lighting creates a strong 3D illusion — facets facing the light are brighter, those facing away are darker
- Thin facet lines radiate from each cell center to the edges, reinforcing the gem-cut appearance
- Lower seed counts produce large, prominent crystal faces; higher counts give a fine mosaic of tiny facets
- Adjusting the light angle dramatically changes the perceived depth and dimensionality of the pattern
