# Topographic

Simplex noise rendered as contour lines, mimicking the elevation isolines found on topographic maps.

## Algorithm

The pattern generates a continuous height field and then extracts contour lines by detecting level-set crossings between adjacent pixels:

1. A 2D simplex noise function is initialized from the deterministic seed.
2. A random offset `(offsetX, offsetY)` (0–1000) is chosen per seed to shift the noise domain, ensuring different terrain per seed.
3. **Height field computation**: A `Float32Array` stores the fBm value for every pixel. Each evaluation uses `octaves` layers with lacunarity 2 and persistence 0.5.
4. The noise range `[-1, 1]` is divided into `contourLevels` evenly spaced bands. The step size is `2 / contourLevels`.
5. **Contour detection**: For each pixel, its quantized level is compared with its right and bottom neighbors. If `floor((value + 1) / step)` differs between a pixel and either neighbor, the pixel lies on a contour boundary.
6. Contour pixels are colored using the foreground palette (cycling by level index with a seed-based rotation offset). Non-contour pixels remain the background color.
7. The result is written via `ImageData` for efficient pixel-level rendering.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Noise Scale | 0.001 – 0.015 | 0.004 | Spatial frequency of the height field. Lower = broader terrain features |
| Contour Levels | 4 – 32 | 16 | Number of elevation bands. More levels = denser contour lines |
| Octaves | 2 – 8 | 5 | fBm octaves for the height field. More octaves add terrain detail |

## Visual Characteristics

Topographic produces clean, thin contour lines on a solid background — closely resembling real cartographic maps. Lines cluster together on steep noise gradients (like closely spaced contours on a cliff face) and spread apart in flat regions. The multi-color cycling gives each elevation band a distinct hue, making the layered structure visible at a glance. The overall aesthetic is precise and technical, with an elegant mathematical beauty.
