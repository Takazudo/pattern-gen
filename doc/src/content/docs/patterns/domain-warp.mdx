# Domain Warp

Layered noise with domain warping creates swirling, psychedelic organic shapes rendered as colored regions — a technique popularized by Inigo Quilez.

## Algorithm

Domain warping feeds the output of one noise evaluation back as the input coordinates of another, compounding distortion across multiple layers:

1. A 2D simplex noise function is initialized from the deterministic seed.
2. The canvas is iterated pixel-by-pixel (in 2×2 blocks for performance).
3. For each block at position `(x, y)`, the algorithm applies two successive warp layers:
   - **First warp**: Two independent fBm (fractional Brownian motion) evaluations produce offset vectors `(warpX, warpY)`. The second evaluation uses offset seeds `(+5.2, +1.3)` to decorrelate the axes. Both are scaled by `warpStrength * noiseScale`.
   - **Second warp (domain warp)**: Two more fBm evaluations sample at the original position *plus* the first warp offset, with different offset seeds `(+1.7/+9.2)` and `(+8.3/+2.8)`. This creates the characteristic swirling distortion.
4. A final fBm evaluation at the double-warped coordinates produces the output value.
5. The value is normalized from `[-1, 1]` to `[0, 1]` and mapped to a palette color index.

Each fBm call uses the configured number of `octaves`, with the standard lacunarity of 2 and persistence of 0.5. The multi-layer warping creates structures that cannot emerge from simple noise alone — deep folds, vortex-like swirls, and amoeba shapes.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Noise Scale | 0.001 – 0.02 | 0.004 | Base spatial frequency of the noise. Lower = larger structures |
| Warp Strength | 20 – 200 | 80 | How far the domain is displaced. Higher = more dramatic distortion |
| Octaves | 2 – 8 | 4 | Number of fBm octaves. More octaves add finer detail to each layer |

## Visual Characteristics

Domain Warp produces large-scale flowing regions of color with complex, folded boundaries. The double-warping creates organic shapes reminiscent of fluid dynamics, geological formations, or microscopic cell structures. Unlike simple noise patterns, the recursive distortion generates deep visual complexity — nested swirls within swirls. The palette maps continuously across regions, producing smooth color gradients where warp boundaries are gentle and sharp transitions at fold lines.
