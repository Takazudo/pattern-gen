# Meander

Greek key meander pattern with nested rectangular spirals arranged in concentric frames with multiple border and fill styles.

## Algorithm

The meander (also called Greek key or Greek fret) is one of the most enduring decorative motifs in Western art, originating in ancient Greek pottery and architecture. It represents an unbroken line that folds back on itself in a continuous, labyrinthine path — symbolizing infinity or the eternal flow of life.

The algorithm works as follows:

1. **Frame construction**: One or more concentric rectangular frames are drawn around the canvas, spaced by `unit * 5`. Each frame gets alternating colors (key color vs. accent color) and a rectangular border stroke.

2. **Border pattern rendering**: Along each side of each frame (top, bottom, left, right), a border pattern is drawn within a band 3 units high. Four border styles are available:

   - **Classic**: Traditional Greek key motifs — alternating mirrored rectangular spirals that wind inward. At low complexity (≤2), a simple single-turn key is drawn. At higher complexity (3–5), the key winds multiple times with progressively smaller inset rectangles.

   - **Running**: Independent rectangular spirals that each wind clockwise inward. The number of turns matches the complexity parameter. Each spiral unit sits side-by-side along the border band.

   - **Wave**: Sinusoidal S-curves drawn along the border. Multiple wave layers stack at evenly spaced offsets within the band height, with decreasing amplitude. The wavelength tightens with higher complexity.

   - **Labyrinth**: A maze-like grid of randomly placed walls. Each cell has a chance of getting right and bottom walls proportional to `0.3 + complexity * 0.1`. Occasional short stub lines create dead-end details.

3. **Interior fill**: The area inside all frames is filled with one of four styles:
   - **Key Tiles**: Repeating Greek key spiral tiles (a rectangular spiral that winds from corner to center).
   - **Zigzag**: Rows of zigzag lines with slight random horizontal offset.
   - **Spiral**: A single Archimedean spiral expanding from the center of the interior region.
   - **Empty**: No interior fill.

4. **Decorative accents**: When border decoration is enabled, small dots are placed at key corner points collected during border drawing. A subtle outer shadow stroke adds depth to the outermost frame.

## Parameters

| Parameter | Range | Default | Description |
|-----------|-------|---------|-------------|
| Unit Size | 10–80 | 40 | Grid division count for the base unit size. Lower values produce larger units and bolder patterns. |
| Line Width | 1–10 | 4 | Stroke width in pixels for all drawn lines. |
| Frame Count | 1–5 | 2 | Number of concentric rectangular frames. |
| Border Style | Classic, Running, Wave, Labyrinth | Classic | Which border pattern to draw along the frames. |
| Fill Style | Key Tiles, Zigzag, Spiral, Empty | Key Tiles | How the interior area within all frames is filled. |
| Key Complexity | 1–5 | 2 | Number of turns in each key motif (Classic/Running) or density factor (Wave/Labyrinth). |
| Border Decoration | On/Off | Off | Whether to draw decorative dots at corner intersection points. |

## Visual Characteristics

The meander pattern has a strongly architectural quality with its rectilinear frames and geometric borders. The Classic border style is instantly recognizable as the Greek key motif seen on pottery, temple friezes, and neoclassical architecture. Running spirals create a more dynamic, scrolling rhythm. The Wave style softens the rigid geometry with organic curves. The Labyrinth style produces a randomized, maze-like texture.

The frame count parameter transforms the pattern from a simple bordered rectangle (1 frame) to a richly nested design (5 frames) with alternating colors creating a sense of depth. Key complexity dramatically affects density — at complexity 1, keys are simple L-shapes, while at 5 they become tight multi-turn spirals.

The interior fill styles provide contrasting character: Key Tiles extend the meander aesthetic inward, Zigzag adds diagonal energy, and Spiral introduces a contrasting curvilinear form. Empty fill leaves a clean open center suitable for overlaying text or imagery.
