# Preset Sizes

The Preset Size Chooser provides common image dimensions for popular platforms.
When you select a preset, the canvas switches to **Fixed** aspect mode with the
preset's exact width and height.

## Using Presets

1. Open the **Selection Overlay** (the generate/compose toolbar)
2. Click the **Presets** button in the aspect ratio bar
3. Browse by category or view all presets
4. Click a preset to apply — the canvas immediately switches to Fixed mode with the selected dimensions
